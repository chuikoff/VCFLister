## [2.1] – 2026-06-23

### Added
- Статусбар под списком контактов: общее количество + текущая позиция (`Контактов: N (k/N)`).
- Вертикальный скроллбар для правой панели карточки (hRightScroll). Теперь можно прокручивать большое фото вместе с длинным текстом контакта.
- Показ всех контактов, включая полностью пустые (раньше прятались).

### Changed / Improved
- EDIT в правой панели теперь явно имеет `WS_VSCROLL | WS_HSCROLL` — скроллбар появляется когда нужно.
- Фото-URL (PHOTO:https://...) теперь отображаются в текстовом блоке (не скрываются).
- Улучшена обработка `itemN.PHOTO`, `X-FCENCODED-*` и других кастомных полей из vCard 4.0 (в т.ч. примеры из ez-vcard).
- Пустые карточки показываются как "(пустая карточка)" / "(empty card)" в списке.

### Fixed
- Исправлено скрытие PHOTO-URL и утечка base64-данных в текст.
- Синхронизация индексов при показе пустых контактов.
- Мелкие улучшения стабильности скролла и layout'а.

### Build
- Версия поднята до 2.1.
- Бинарники: `VCFLister.wlx64` (x64), `VCFLister.wlx` (Win32).

## [2.0] – 2026-06-18

### Added
- Полноценная поддержка сложных vCard (Apple/macOS, OPPO и др.): X-ABLABEL, X-SOCIALPROFILE, X-ABDATE, X-ABRELATEDNAMES, X-ALTBDAY, IMPP, X-ANDROID-CUSTOM, phonetic имена и т.д.
- Улучшенное отображение: кастомные метки приклеиваются к полям (вместо отдельных "Label: _$!<...>!$_"), очистка лишних `;` в N/ADR, красивый разбор Android Custom Events.
- Пропуск полностью пустых контактов (только BEGIN/END/VERSION) — они больше не попадают в список и не сдвигают индексы.
- Фото выравнивается по левому верхнему краю (top-left) с сохранением пропорций.
- Фокус в поле данных (EDIT) больше не ломает переключение плагинов и режимов просмотра (HEX/PlainText) в Total Commander.

### Fixed
- Проблема с пустыми карточками: теперь синхронно фильтруются и в ParseVCard, и в SplitVCardBlocks → правильные контакты и фото.
- Убрана принудительная полоса прокрутки в панели деталей (WS_VSCROLL/WS_HSCROLL на EDIT). Появляется только когда текст не влезает.
- Улучшена надёжность загрузки фото (восстановлена схема владения HGLOBAL для GDI+ Bitmap, более терпимый Base64Decode + предочистка).
- Мерцание при прокрутке и обновлении фото: WS_CLIPCHILDREN, двойная буферизация в окне фото, InvalidateRect с FALSE.
- Фокус после клика мышью в данные контакта: EditSubclass + ES_NOHIDESEL + возврат фокуса на главное окно плагина.
- Корректная обработка ENCODING=B и BASE64, folded строк v2.1/v3.0, QP, charset.

### Changed
- Рефакторинг: общие утилиты вынесены в vcf_utils.* (убраны дубликаты), удалён мёртвый PhotoDrawer.
- Размер фото увеличен примерно в 2 раза (500 → 1000).
- Улучшена очистка полей (N/ADR без лишних `;`, пропуск пустых FN/N).
- Локализация и BuildFromRawBlock доработаны для кастомных полей (X-*, itemN.*).
- Версия поднята до 2.0 (значительные улучшения отображения, стабильность, UX).

### Build
- Конфигурация: x64 Release → `VCFLister.wlx64`, Win32 Release → `VCFLister.wlx`.
- Проверено в Total Commander 11.54 (Lister) и выше.
- Генерация версии через gen_version.ps1 (git tag + build counter).
