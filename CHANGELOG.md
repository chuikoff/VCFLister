## [Unreleased] – последние 3 дня

### Added
- Поиск по всем полям (F7/F3/Shift+F3) через `ListSearchTextW` + `VCFView_SearchEx`.
- Автопрокрутка левого списка к выбранному/найденному контакту.
- Контекстное меню «Копировать значение» для полей.

### Fixed
- Регистронезависимый поиск по кириллице и Unicode (перевод регистра через `CharLowerBuffW`).
- Парсер vCard 2.1: корректная склейка переносов soft wrap и `ENCODING=QUOTED-PRINTABLE`
  (включая `CHARSET=UTF-8`), убраны хвостовые `=` в именах.
- Игнор «пустых» карточек BEGIN/END без данных; в списке больше не рисуется `Email:` без значения.
- Безопасность Clipboard/GDI: проверки `GlobalLock` (убран C6387).
- Экспорт функций через `listplug.def` (гарантированно видит TC); устранены конфликты
  с `listplug.h` (плагин самодостаточен).

### Changed
- Чтение .vcf с авто-детектом кодировки: UTF-16LE BOM, UTF-8 (строг.), CP1251, OEM, ACP.
- Упрощена верстка правой панели (подписи не наезжают; номера/поля не делаем кликабельными).
- Исходники перекодированы в UTF-8 (без BOM), чтобы комментарии на GitHub отображались корректно.

### Build
- Конфигурация: x64 Release → `VCFLister.wlx64`.
- Проверено в Total Commander 11.54 (Lister).
