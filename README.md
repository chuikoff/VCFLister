# VCFLister

**VCFLister** — плагин Lister (`.wlx64`) для [Total Commander](https://www.ghisler.com/), позволяющий удобно просматривать контакты из файлов формата **VCF/vCard**.

## Возможности

- Поддержка формата **vCard 3.0** (в т.ч. `itemN.TEL`, `itemN.EMAIL`).
- Отображение списка контактов (слева) и подробной информации по выделенному контакту (справа).
- Поддержка фото (через GDI+), автоматическое масштабирование, DPI-aware.
- Отображаются все основные поля:
  - Имя (FN, N)
  - Телефоны (с типами HOME/WORK/MOBILE и т.д.)
  - Email-адреса
  - Организация (ORG), должность (TITLE)
  - Дата рождения (BDAY)
  - Адреса (ADR)
  - Примечания (NOTE)
  - Ссылки (URL)
- Поддержка поиска по всем полям (Ctrl+F / F3 / Shift+F3).
- Полная поддержка сложных vCard (Apple, OPPO и др.): X-ABLABEL, социальные профили, связанные имена, альтернативные даты, Android Custom Events и т.д. (метки приклеиваются к полям, без сырого "Label: _$!<...>!$_").
- Показ всех контактов, включая пустые (отмечаются как "(пустая карточка)").
- Фото по левому верхнему краю.
- Стабильное переключение плагинов и режимов просмотра (даже после клика в данные).
- Улучшенная очистка полей (N, ADR без лишних `;`), надёжная работа с фото и фокусом.

## Установка

1. Скопируйте файл плагина в папку `Plugins\Wlx` Total Commander (или любую подпапку).
2. В Total Commander откройте любой .vcf файл — плагин подключится автоматически.
3. Для 64-битной версии Total Commander используйте `VCFLister.wlx64`.
4. Для 32-битной версии — `VCFLister.wlx`.
